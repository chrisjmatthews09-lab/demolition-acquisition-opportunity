        - Key Metrics (EBITDA Growth, Multiple, DSCR): Goal: Inform -> Viz: Dynamic Stats (Large text cards) -> Interaction: None -> Justification: Immediately grabs attention with the most critical deal highlights. Method: HTML/Tailwind.

        - National Market Growth: Goal: Change -> Viz: Line Chart -> Interaction: Tooltip on hover -> Justification: Clearly shows the positive market trend over time, establishing a stable industry backdrop. Library: Chart.js (Canvas).

        - Revenue Focus: Goal: Compare -> Viz: Donut Chart -> Interaction: Tooltip on hover -> Justification: Simple, effective way to show the company's heavy concentration in the desirable commercial sector. Library: Chart.js (Canvas).

        - Competitive Landscape: Goal: Organize -> Viz: Diagram (Flexbox columns) -> Interaction: Hover effects -> Justification: A structured HTML/CSS diagram clearly explains the "barbell" market concept without needing complex graphics, highlighting the target's strategic position.         - Financial Growth (Revenue & EBITDA): Goal: Compare/Change -> Viz: Combined Bar/Line Chart -> Interaction: Tooltip on hover -> Justification: Effectively visualizes both the scale of revenue (bar) and the faster growth of profitability (line) in a single, compelling view. Library: Chart.js (Canvas).

        - EBITDA Margin Expansion: Goal: Change -> Viz: Area Chart -> Interaction: Tooltip on hover -> Justification: Emphasizes the significant and sustained increase in profitability over time, a key value driver. Library: Chart.js (Canvas).

        - Valuation vs. Benchmarks: Goal: Compare -> Viz: Floating Bar Chart -> Interaction: Tooltip on hover -> Justification: Provides clear context for the asking price, showing it falls within a reasonable range, making the valuation feel justified and attractive. Library: Chart.js (Canvas).

        - DSCR: Goal: Inform -> Viz: Gauge Chart (customized Donut) -> Interaction: None -> Justification: A gauge provides an immediate visual assessment of the company's ability to service debt, a critical metric for any investor. Method: Chart.js (Canvas) with custom text overlay.

        - Risks: Goal: Organize -> Viz: Table -> Interaction: Gemini-powered button to generate DDQs -> Justification: A simple table is effective, enhanced with an AI tool to generate deeper, actionable questions for investors. Method: HTML/Tailwind & Gemini API.

        - Growth Levers: Goal: Organize -> Viz: Interactive Tags -> Interaction: Click to generate a strategic plan via Gemini -> Justification: Turns static tags into an interactive exploration tool for post-acquisition strategy. Method: HTML/Tailwind & Gemini API.

        async function callGeminiAPI(payload, retries = 3, delay = 1000) {

            const apiKey = "YOUR_API_KEY_HERE"; // 👈 PASTE YOUR GEMINI API KEY HERE

            const apiUrl = `https://generativelanguage.googleapis.com/v1beta/models/gemini-2.5-flash-preview-05-20:generateContent?key=${apiKey}`;

            

            for (let i = 0; i < retries; i++) {

                try {

                    const response = await fetch(apiUrl, {

                        method: 'POST',

                        headers: { 'Content-Type': 'application/json' },

                        body: JSON.stringify(payload)

                    });

                    if (!response.ok) {

                        if (response.status === 429 || response.status >= 500) {

                            throw new Error(`API call failed with status: ${response.status}. Retrying...`);

                        }

                        throw new Error(`API call failed with status: ${response.status}`);

                    }

                    return await response.json();

                } catch (error) {

                    if (i === retries - 1) {

                         console.error("Gemini API call error after all retries:", error);

                         return { error: `Failed to fetch from Gemini API. ${error.message}` };

                    }

                    await new Promise(res => setTimeout(res, delay * Math.pow(2, i)));

                }

            }

        }
